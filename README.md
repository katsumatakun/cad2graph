# cad2graph
### The purpose of this project is to convert a cad file to a graph object in an automated way.

pyparsing==2.3.0
python-dateutil==2.7.5
scipy==1.2.0
Shapely==1.6.4.post2
six==1.12.0


#### Demo
Run demo.py will create the node list, the edge list, the stair node list, and elevator node list as well as an image showing a graph overlaid on CAD file.
